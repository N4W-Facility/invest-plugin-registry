# Security Policy

The inclusion of a plugin in the InVEST plugins registry does not imply or guarantee
anything about the plugin’s quality, suitability, or security, and the Natural Capital
Alliance reserves the right to remove any plugin from the registry that is believed to
be a security risk. If you believe a plugin poses a security risk, please go to the
Security and Quality tab of the Registry GitHub Repo and use the
“Report a vulnerability” button to file a report.

Security and Quality tab: https://github.com/natcap/invest-plugin-registry/security
